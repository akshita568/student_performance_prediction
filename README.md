# Student Performance Prediction Platform
## What does it do?
This is an AI-powered prediction platform that analyzes how different factors affect student performance. The model predicts student grades based on features such as gender, ethnicity, race, parental education level, lunch type, test preparation course completion, and other academic and demographic factors.

## Features 
- predicts student performance using machine learning
- analyzes the impact of multiple demographic and educational factors
- user friendly interface for entering student details
- provides data-driven insights into academic outcomes
